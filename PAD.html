Puzzle and Dragons Monster Filter


The completed application can be viewed on Heroku here.

Overview: This app is based on the popular "Puzzle and Dragons" phone game. The goal of this application is to give a user personalized information on select Monsters from the game. There are thousands of Monster cards in the game itself, but this application aims to gives information on only a small subset of Monster cards - those that are owned and managed by the user. The user can interact with the webpage in a few different ways: through the filter dropdowns, through the seach bar, or through various buttons. These elements work together to display select Monster data on the page.

Sample of Monster Cards and Details

Design Challenges and Decisions: During the project's conception, I'd originally wanted a way to add and delete new Monster data from the database. This would allow users to personalize the application based on the Monster cards they themselves owned. However, I ultimately scrapped implementing such a feature, given both my time constraints and the simple scope of the project assignment.

Conclusion: This project introduced me to the React JavaScript library, and the experience went well enough that I intend to utilize React in future projects. There are a lot of features that I did not have the time (or means) to implement in this application, but I hope to build off the JavaScript knowledge I gained from it all the same.
