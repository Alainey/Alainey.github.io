Betrayal at House on the Hill Online

Back End Development - Front End Development - UI/UX

Java - Javascript - HTML - CSS

An online version of the popular multiplayer tabletop game: Betrayal at House on the Hill. The goal of this project was to create an interactive version of Betrayal at House on the Hill that would allow users to play with one another from the comfort of their own home.

This project was a collaboration between myself and three other Software Engineering students. I worked on the front-end side of this program. Additionally, I spent a majority of my time working on the integration between our front-end and back-end code.
