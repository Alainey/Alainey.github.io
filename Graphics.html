Virtual Reality Atmospheric Scene


The code for this project can be foun don my github here.

Overview: This project is a collaborative piece designed and programmed by myself and a fellow classmate. This project is a Vive-compatible simulation that places the user in the middle of a densely foggy lake. Our goal was to create an atmospheric scene inspired by natural hot springs, with realistic water movement and the inclusion of fog/steam.



Screenshot of the Completed Scene

Preliminary Trees (Generated with Lindenmayer Systems)

Design Challenges and Decisions: Time was one of our biggest constraints with this project. We had a lot of ideas that we wanted to incorperate into our final project, but didn't have nearly enough time to add them all.


Screenshot of the Completed Scene

Conclusion: There are many aspects of this project that I wish we had improved on. The L-system trees are unrealistic due to being made up of only lines, and - although they're creepy as they are - they don't really fit with the rest of the scene. Adding greater width to the lower nodes, as well as forming the branches with scaled cylinders as opposed to lines, would have been two ways in which we would have done this.

Additionally, all of our land masses are made out of scaled up spheres, and I would have likes to add a bit more detail to these as well. We were considering bump-mapping and/or simple terrain generation for our islands, but we never had time to implement either.
